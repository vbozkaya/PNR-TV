# PNR TV - Gizlilik Politikası

Bu klasör GitHub Pages için gizlilik politikası sayfalarını içerir.

## GitHub Pages Kurulumu

1. **Repository Ayarları:**
   - GitHub repository'nize gidin
   - Settings > Pages bölümüne gidin
   - Source: "Deploy from a branch" seçin
   - Branch: `main` (veya `master`)
   - Folder: `/docs` seçin
   - Save'e tıklayın

2. **URL:**
   - GitHub Pages URL'niz şu formatta olacak:
   - `https://yourusername.github.io/pnr-tv/privacy-policy.html`
   - veya custom domain kullanıyorsanız:
   - `https://yourdomain.com/privacy-policy.html`

3. **Güncelleme:**
   - `docs/privacy-policy.html` dosyasındaki GitHub repository URL'sini güncelleyin
   - (Satır 195 civarında: `https://github.com/yourusername/pnr-tv`)

## Dosyalar

- `index.html` - Ana sayfa (dil seçimi)
- `privacy-policy.html` - Türkçe gizlilik politikası
- `privacy-policy-en.html` - İngilizce gizlilik politikası

## Google Play Console'da Kullanım

Google Play Console'da gizlilik politikası URL'si olarak şunu kullanın:
```
https://yourusername.github.io/pnr-tv/privacy-policy.html
```

veya İngilizce için:
```
https://yourusername.github.io/pnr-tv/privacy-policy-en.html
```
